Bible.io

  • Date: Februrary 25, 2019
  • Contributors: Yash Nagpal, Keith Fernandes, Mark Bedaywi, Tyler Wong
  • Github: N/A

Description:
Created for the DPCDSB Game Making Competition (Bible Story Themed), this is an online 'io' game comprised of multiple minigames. The minigames rotate every few seconds and a global scoreboard keeps track of each player's ID and score. There are 3 minigames where the instructions for each are displayed before the game starts. The game uses Javascript, p5.js, Node.js and socket.io for all games. I was in charge of designing and programming a framework to combine all games programmed by my teammates. "Bible.io" won the "Most Technical Game" award at the competition. The game is currently playable here (Be sure to turn down your volume before playing though, the music is quite loud).
